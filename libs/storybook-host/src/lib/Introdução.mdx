## Sobre

Este Design System foi criado para:

- **Centralizar tokens e componentes reutilizáveis**
- **Facilitar a consistência visual** em projetos Angular
- **Melhorar a produtividade** do desenvolvimento frontend

### Features Principais

- Tokens reutilizáveis (Cores, Tipografia, Espaçamentos)
- Componentes modulares
- Integração com o monorepo NX
